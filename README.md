# Hospital Inventory Management System

A real-time hospital inventory management system built with the MERN stack.

## Features
- Real-time ventilator tracking
- Inventory management across 50+ hospitals
- Ambulance routing system
- Live updates using Socket.io

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
- Socket.io for real-time updates
